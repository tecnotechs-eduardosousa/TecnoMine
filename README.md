# TecnoMine

TecnoMine é um pequeno sistema com a finalidade de acelerar processos do dia a dia para os desenvolvedores da TecnoTech Sistemas que utilizam o RedMine.

### Disclaimer:
Tendo em vista que o TecnoMine é um sistema direcionado para processos internos da TecnoTech Sistemas, este software é de direito desta empresa, não de quaisquer funcionários que tenham criado e/ou contribuído para tal projeto.
